DevOps Tools Directory

Your go-to resource for DevOps tools.

CI/CD Tools

  • Jenkins - Open-source automation server.
  • CircleCI - Continuous integration and delivery platform.
  • Travis CI - CI service used to build and test projects hosted on GitHub.
  • GitLab CI/CD - Built-in CI/CD features for GitLab repositories.

Configuration Management

  • Ansible - Simple IT automation tool.
  • Puppet - Software configuration management tool.
  • Chef - Automation platform for configuring and managing servers.
  • Terraform - Infrastructure as code tool for building, changing, and versioning infrastructure.

Monitoring & Logging

  • Prometheus - Open-source monitoring and alerting toolkit.
  • ELK Stack - Elasticsearch, Logstash, and Kibana for logging and visualization.
  • Grafana - Open-source platform for monitoring and observability.
  • Datadog - Monitoring and analytics platform for cloud applications.

Containerization & Orchestration

  • Docker - Platform for developing, shipping, and running applications in containers.
  • Kubernetes - Container orchestration platform.
  • OpenShift - Kubernetes-based container platform.
  • Nomad - Scheduler and orchestrator for managing containers.
